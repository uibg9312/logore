# Logore

Logore is an AI-powered logo processing tool built with Next.js, Tailwind CSS, and Google Gemini.

## Features
- **Background Removal**: Automatically removes the background from uploaded logos.
- **AI Analysis**: Uses Gemini 2.5 Flash to analyze the logo and understand its structure.
- **Variations**: Generates monochrome, outline, and vector variations of the logo.
- **Export Options**: Download variations in PNG or SVG formats.

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Add your Gemini API key in `.env.local`.
4. Run `npm run dev` to start the development server.

## Technologies
- Next.js
- Tailwind CSS 4
- Google Gemini (Flash & Flash Image)
- Framer Motion
